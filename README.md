A Powershell Script (with GUI) which handles different aspects of Virt-A-Mate .var files

            Pack       Compresses folders back to .var files.`nRestores original timestamps from .orig files.
            Unpack     Extracts .var files.
            Sanitize   Disables DLLs inside .var files.
            Desanitize Restores disabled DLLs inside .var files.
            Repair     Extracts/Repacks corrupt archives.
            Unify      Merges content to 'anonymous.merged.1.var' (no versioning, no nothing, just files combined).
            Restore    UNDO: Restores .orig files, deletes changes.
            View       Lists file content without extracting.

Warning!
- Use at your own risk!
- make backup copies!
